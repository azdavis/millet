<!-- @config language.successor-ml.record-update = true -->

Record update syntax was used, but the fields being updated weren't present in the original record.

```sml
val x = { a = 1, b = "hey" }
val y = { x where c = false }
(**     ^^^^^^^^^^^^^^^^^^^^^ record update provided extra rows *)
```

## To fix

Only update fields that were in the original record.

```sml
val x = { a = 1, b = "hey" }
val y = { x where a = 2 }
```
